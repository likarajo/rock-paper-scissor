Libraries:

  • PIL
    Install PIL (Python Imaging Library)
    >>> pip install PIL

  • tkinter
    Install tkinter library (for GUI development)

    >>> pip install tkinter

  • random
    Install random library (for generating random numbers)

    >>> pip install random

Run Docker Container"

Creating Executable:

  • pyinstaller
    Install pyinstaller library into %PYTHON_HOME%\Lib\site-packages
    >>> pip install pyinstaller
    This also installs pypiwin32 utilities library of python extension for windows

Create the executable file using the following command:
> pyinstaller.exe –onefile –windowed game.py

Inputs:

User selects ROCK, PAPER, or SCISSOR by clicking on the corresponding button.

Output:

Computer's choice is made using random library, and accordingly result is obtained with below rule:
ROCK beats SCISSOR; PAPER beats ROCK; SCISSOR beats PAPER.
